N = 0일 때 
다 비운 경우의 수 1
= 1

N = 1일 때 
다 비운 경우의 수 1
하나씩 채워나가는 경우의 수 2
= 3

N = 2일 때
다 비운 경우의 수 1
하나씩 채워나가는 경우의 수 4
1, 4칸 2,3칸 채운 경우의 수 2
= 7

여기까지 점화식을 세워본다면 dp[i] = dp[i-1]*2 + dp[i-1]정도가 될 듯하다.

N  = 3일 때
다 비운 경우의 수 1 
하나씩 채워나가는 경우의 수 6
1, 4칸 채우고 2, 3칸 채운 경우의 수 2
1, 5칸 채우고 2, 6칸 채운 경우의 수 2
1, 4, 5칸, 2, 3, 6칸 채운 경우의 수 2
1, 6칸 채우고 2, 5칸 채운 경우의 수 2
4, 5칸 채우고 3, 6칸 채운 경우의 수 2
= 17개

점화식 확인 : dp[3] = dp[2] * 2 + dp[1] = 17
즉 세웠던 점화식이 맞다는 걸 확인 가능하다.
